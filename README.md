## Start

사전에 설치 및 환경변수 설정이 필요합니다. `.env.example` 파일을 참고하여 `.env.development` 파일을 생성하여 실행해주세요.

궁금한 정보가 있다면 상단에 있는 이메일로 연락을 부탁드립니다.

```
npm i
npm run dev
```
